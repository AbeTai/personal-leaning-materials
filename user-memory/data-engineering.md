# Data Engineering / データマネジメント関連の知識

## 理解している概念・技術

### データ分析（SQL/BigQuery）
- **理解レベル**: 中級（SQLやBigQueryでのデータ分析は業務で実施している）
- **詳細**: 分析側の立場でデータを使うことは多いが、データ基盤（データレイク/DWH/マート）の設計・整備を自分で行った経験はまだない。
- **最終更新**: 2026-07-14

### データマネジメント三層構造（データレイク/DWH/マート）
- **理解レベル**: 初級（本教材で学習）
- **詳細**: データソース→データレイク（そのままコピー）→DWH（共通化）→データマート（目的別加工）という段階と、それぞれの役割の違いを学習した。
- **最終更新**: 2026-07-14

### メタデータ管理
- **理解レベル**: 初級（学習中、重点領域）
- **詳細**: ビジネス/テクニカル/オペレーショナルの3分類、BigQueryでのカラム説明管理、dbt-osmosisによる説明伝播（`yaml document`/`yaml refactor`）を学習した。実務での導入は未経験。
- **最終更新**: 2026-07-14

### セマンティックレイヤー（dbt/Looker）
- **理解レベル**: 中級（使ったことがある）
- **詳細**: dbtやLookerのセマンティックレイヤー機能は使用経験あり。ファクト・ディメンションの組み合わせ定義や「指標の意味を統一する」という設計思想は本教材でさらに深掘りした。
- **最終更新**: 2026-07-21

### オントロジー・統制語彙（シソーラス/タクソノミー/フォークソノミー）
- **理解レベル**: 初級（本教材で学習）
- **詳細**: 用語同士が似ていて区別がつかなかったが、トップダウン統制語彙（シソーラス・オントロジー・タクソノミー）とボトムアップのフォークソノミーという整理軸を学習した。
- **最終更新**: 2026-07-21

### グラフDB・ナレッジグラフ
- **理解レベル**: 初級（重点領域として学習）
- **詳細**: ノード・エッジの基本構造、リレーショナルDBとの違い、BigQuery Graph（サーバレス）の登場、隣接行列による大規模データ処理、消費者-飲食店の支払いグラフ事例を学習した。実務での構築・分析経験はまだない。
- **最終更新**: 2026-07-21
