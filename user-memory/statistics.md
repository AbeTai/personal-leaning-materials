# Statistics / 因果推論・A/Bテスト関連の知識

## 理解している概念・技術

### A/Bテスト（基礎）
- **理解レベル**: 中級（業務でA/Bテスト分析の実施経験あり）
- **詳細**: t検定などの基本的な検定は理解し、業務で実施したことがある。
- **最終更新**: 2026-07-13

### DID（差分の差分析, Difference-in-Differences）
- **理解レベル**: 中級（使ったことがある）
- **詳細**: ベースライン(Y0)とフォローアップ(Y1)の差分 `(Y1-Y0) = a + b・Treat + 誤差` を用いて個体差由来のノイズを取り除く設計は理解・使用経験あり。
- **最終更新**: 2026-07-13

### ANCOVA（共分散分析, Analysis of Covariance）
- **理解レベル**: 初級〜中級（学習中）
- **詳細**: `Y1 = a + b・Treat + θ・Y0 + 誤差` の形でベースラインを共変量として扱う設計。DIDとの使い分け（自己相関ρの高低による選択基準）を学習した。
- **最終更新**: 2026-07-13
